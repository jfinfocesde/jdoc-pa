## Actividad 2: Crear un proyecto de Python en Visual Studio Code

1.  Preparación:

    - Asegúrate de tener instalado Python en tu ordenador.
    - Descarga e instala Visual Studio Code.

2.  Crear el proyecto:

    - Abre Visual Studio Code.
    - Selecciona Archivo > Nuevo > Carpeta.
    - Nombra la carpeta como tu proyecto (por ejemplo, "Actividad2").
    - Abre la carpeta en Visual Studio Code.

3.  Agregar un archivo al proyecto:

    - Haz clic derecho en la carpeta del proyecto.
    - Selecciona Nuevo archivo.
    - Nómbralo como `app.py`.

4.  Implementar algoritmos de lógica de programación en Python:

    - Abre el archivo `app.py`.
    - Escribe tu código Python, incluyendo:
      - Variables: Usa diferentes tipos de datos como `int`, `str`, `float`, etc.
      - Operadores: Realiza operaciones matemáticas, lógicas y comparativas.
      - Estructuras condicionales: Usa `if`, `elif` y `else` para ejecutar diferentes acciones según las condiciones.
      - Bucles: Usa `for` y `while` para ejecutar bloques de código de forma repetitiva.

5.  Imprimir datos por consola:

    - Usa la función `print()` para mostrar mensajes, variables y resultados en la consola.

6.  Subir el proyecto a un repositorio remoto en GitHub:

    - Crea una cuenta en GitHub.
    - Crea un nuevo repositorio en GitHub.
    - Sube los cambios al repositorio remoto.

7.  Requerimientos:

- Usa las indentaciones correctamente.
- Agrega comentarios para explicar tu código.
- Prueba tu código antes de subirlo a GitHub.

8. Entrega el enlace del reposorio al docente
