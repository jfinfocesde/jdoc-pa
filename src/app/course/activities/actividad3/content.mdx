### Ejercicios con bucles `while` :

1\. Adivinar un número:

El programa pide al usuario que introduzca un número entre 1 y 100. El programa entonces genera un número aleatorio entre 1 y 100 y le dice al usuario si su número es mayor, menor o igual al número aleatorio. El programa continúa hasta que el usuario adivina el número aleatorio.

```python
numero_aleatorio = random.randint(1, 100)
numero_usuario = 0

while numero_usuario != numero_aleatorio:
    numero_usuario = int(input("Introduce un número entre 1 y 100: "))

    if numero_usuario > numero_aleatorio:
        print("El número es menor.")
    elif numero_usuario < numero_aleatorio:
        print("El número es mayor.")

print("¡Felicidades! Has adivinado el número.")

```

2\. Calcular la suma de los números pares del 1 al 100:

El programa calcula la suma de todos los números pares entre 1 y 100.

```python
suma = 0
numero = 1

while numero <= 100:
    if numero % 2 == 0:
        suma += numero

    numero += 1

print("La suma de los números pares del 1 al 100 es:", suma)

```

3\. Contar las vocales en una cadena:

El programa pide al usuario que introduzca una cadena de texto y luego cuenta el número de vocales en la cadena.

```python
cadena = input("Introduce una cadena de texto: ")
vocales = 0

for caracter in cadena:
    if caracter in "aeiou":
        vocales += 1

print("El número de vocales en la cadena es:", vocales)

```

4\. Imprimir los números impares del 1 al 50:

El programa imprime todos los números impares del 1 al 50.

```python
numero = 1

while numero <= 50:
    if numero % 2 != 0:
        print(numero)

    numero += 1

```

5\. Factorial de un número:

El programa pide al usuario que introduzca un número y luego calcula el factorial de ese número.


```python
numero = int(input("Introduce un número: "))
factorial = 1

while numero > 1:
    factorial *= numero

    numero -= 1

print("El factorial de", numero, "es:", factorial)

```

1\. Imprimir los números pares del 1 al 100:

2\. Calcular la suma de los cuadrados de los números del 1 al 10:

```python

for numero in range(1, 11):
```

3\. Imprimir una tabla de multiplicar:

```python
numero = int(input("Introduce un número: "))

for i in range(1, 11):
    print(f"{numero} x {i} = {numero * i}")

```

4\. Factorial de un número:

```python
numero = int(input("Introduce un número: "))
factorial = 1

for i in range(1, numero + 1):
    factorial *= i

print("El factorial de", numero, "es:", factorial)

```

5\. Contar las letras en una cadena:

