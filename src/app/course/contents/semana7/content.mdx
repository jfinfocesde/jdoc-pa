# **Express.js: Un Potente Framework para el Desarrollo Web con Node.js**

Express.js es un framework web minimalista, de código abierto y muy popular que se basa en Node.js. Proporciona una base sólida y flexible para crear aplicaciones web y APIs. Estas son sus características clave:

-   **Integración con Node.js:** Aprovecha la potencia y la naturaleza orientada a eventos de Node.js, permitiendo un manejo eficiente de solicitudes concurrentes e interacciones en tiempo real.
-   **Enfoque Minimalista:** Express ofrece un conjunto básico de características sin imponer una estructura específica. Esto te permite personalizar el framework según tus necesidades e integrarlo perfectamente con otras bibliotecas o frameworks.
-   **Enrutamiento:** Define cómo las solicitudes entrantes (URL) se asignan a las funciones controladoras correspondientes que procesan la solicitud y generan una respuesta. Esto facilita la organización clara de la lógica de tu aplicación.
-   **Middleware:** Proporciona un mecanismo poderoso para interceptar y manipular solicitudes y respuestas a lo largo del ciclo de vida de la aplicación. Esto es ideal para tareas como autenticación, registro o análisis de solicitudes.
-   **Motores de Plantillas:** Se integra con varios motores de plantillas como EJS, Pug (anteriormente Jade) o Handlebars, que simplifican el proceso de generar contenido HTML dinámico.

## **Empezando con Express.js**

1.  **Configuración de Node.js y npm:**
    -   Asegúrate de tener Node.js (incluido npm, el Administrador de Paquetes de Node.js) instalado en tu sistema. Puedes descargarlo del sitio web oficial ([https://nodejs.org/en](https://nodejs.org/en)).
2.  **Crea un Nuevo Proyecto:**
    -   Inicializa un nuevo directorio de proyecto usando `npm init -y` (la bandera `-y` crea un archivo `package.json` con configuraciones predeterminadas).
3.  **Instala Express:**
    -   Instala Express como una dependencia usando npm:

        ```bash
        npm install express
        ```

4.  **Aplicación Básica de Express:**
    -   Crea un archivo JavaScript (por ejemplo, `index.js`) y escribe el siguiente código:

        ```javascript
        const express = require('express');
        const app = express();

        app.get('/', (req, res) => {
            res.send('¡Hola desde Express!');
        });

        app.listen(3000, () => {
            console.log('Servidor escuchando en el puerto 3000');
        });
        ```

    -   Ejecuta este código usando Node.js:

        ```bash
        node index.js
        ```

    -   Abre http://localhost:3000 en tu navegador web para ver "¡Hola desde Express!"

## **Conceptos Clave en Express.js**

-   **Enrutamiento:**
    -   Los métodos `app.get()`, `app.post()`, `app.put()`, y `app.delete()` definen cómo se manejan las solicitudes según los métodos HTTP (GET, POST, PUT, DELETE) y las rutas URL.
    -   Puedes definir parámetros de ruta (por ejemplo, `/usuarios/:id`) para capturar valores específicos de la URL.
-   **Middleware:**
    -   Son funciones que se ejecutan en secuencia antes de que una solicitud llegue a su controlador de ruta previsto.
    -   Utiliza middleware para tareas comunes como registro, autenticación o servicio de archivos estáticos.
    -   Express proporciona middleware integrado como `express.static()` para servir archivos estáticos.


# Aplicación usando Node.js, Express y TypeScript:

## Paso 1: Instalación de dependencias

Crea una carpeta para tu proyecto y ejecuta el siguiente comando para inicializar una aplicación [Node.js](https://node.js/) con TypeScript:

```bash
npm init -y
```

Luego, instala las dependencias necesarias:


```bash
npm install express typescript ts-node @types/express @types/node --save
```

## Paso 2: Crear un archivo [tsconfig.json](https://tsconfig.json/)

Crea un archivo tsconfig.json en la raíz de tu proyecto con el siguiente contenido:


```json
  {
    "compilerOptions": {
      "lib": [
        "es2023"
      ],
      "module": "node16",
      "target": "es2022",
      "outDir": "dist",
      "strict": true,
      "esModuleInterop": true,
      "skipLibCheck": true,
      "forceConsistentCasingInFileNames": true,
      "moduleResolution": "node16"
    },
    "include": [
      "src/**/*.ts"
    ],
    "exclude": [
      "node_modules"
    ]
  }
```

## Paso 3: Crear el servidor Express

Crea una carpeta src en la raíz de tu proyecto y agrega un archivo index.ts con el siguiente contenido:

TypescriptCopy

```typescript
import express from "express";

const app = express();
const port = 3000;

app.get("/", (req, res) => {
  res.send("Hola Express con TypeScript!");
});

app.listen(port, () => {
  console.log(`Servidor en ejecución en http://localhost:${port}`);
});
```


